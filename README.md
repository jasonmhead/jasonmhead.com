# jasonmhead.com

Personal website for Jason M Head, hosted on GitHub Pages.

## About

This is a simple, modern personal website built with HTML and CSS. It features:
- Responsive design that works on all devices
- Modern gradient styling
- Sections for About, Projects, and Contact information

This site is automatically published via GitHub Pages at https://jasonmhead.github.io/jasonmhead.com/ (or your custom domain if configured).

- `styles.css` - Customize colors, fonts, and styling
